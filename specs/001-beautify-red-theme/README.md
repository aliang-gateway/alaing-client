## 概述

将 NoneLane Dashboard 前端页面中所有绿色的成功状态元素（按钮、文本、徽章）替换为红色主题配色，实现视觉统一性并保持状态语义清晰。

## 技术决策

### 1. 样式类替换策略
- **选择**: 直接替换 Bootstrap 类名（`btn-success` → `btn-danger`）
- **理由**: 比 CSS 覆盖更简单、更可维护
- **替代方案（已拒绝）**: 保留 success 类用 CSS 覆盖（需要大量 !important）

### 2. CSS 变量处理
- **选择**: 保持 `--bs-success: #f24646` 定义不变
- **理由**: 为未来的自定义组件保留主题色变量
- **实际使用**: HTML/JS 中改用 `danger` 类而非依赖 CSS 变量

### 3. 进度条颜色渐变
- **选择**: 保留渐变逻辑，调整颜色值
- **理由**: 视觉反馈机制必须保留以保持可用性
- **实施**: 低使用率用 `bg-primary`（浅红色），高使用率用 `bg-danger`（深红色）

### 4. 状态徽章语义
- **选择**: 将成功状态改为红色，但通过图标和文字区分状态
- **理由**: 保持语义清晰度，避免混淆
- **实施**: 证书状态徽章已包含图标（bi-check-circle、bi-x-circle）
