## 常见问题排查

### 问题 1: 某些按钮仍显示为绿色

**可能原因**:
- HTML 中有遗漏的 `btn-success` 类
- JavaScript 动态添加的样式类未更新

**解决方法**:
1. 在浏览器开发者工具中检查元素，查看实际应用的类
2. 使用全局搜索 `btn-success`、`text-success`、`bg-success`
3. 检查 JavaScript 代码中的字符串字面量

### 问题 2: 状态语义不清晰

**可能原因**:
- 所有状态都使用红色，无法区分成功/警告/错误

**解决方法**:
- 确认错误状态使用 `bg-danger`（深红）
- 警告状态使用 `bg-warning`（黄/橙）
- 成功状态使用 `bg-danger`（但可添加图标以区分）

### 问题 3: 颜色对比度不足

**可能原因**:
- 红色文字在白色背景上对比度低

**解决方法**:
- 在 `styles.css` 中调整 `--bs-danger` 的色值
- 使用浏览器扩展（如 WAVE）检查可访问性

### 问题 4: 进度条颜色不渐变

**可能原因**:
- JavaScript 逻辑未正确更新

**解决方法**:
- 检查 `app.js:2162` 附近的代码
- 确认条件判断和类名替换正确
